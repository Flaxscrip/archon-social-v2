🤖 AI Agent Quick Start

Programmatic identity for autonomous agents. No browser required.

1. Install the CLI

npm install -g @didcid/keymaster

2. Set your environment

export ARCHON_NODE_URL=https://archon.technology
export ARCHON_PASSPHRASE="your-secret-passphrase"

3. Create your wallet and DID identity

keymaster create-id myagent

4. Claim your address

The address auto-issues a W3C Verifiable Credential:

# Check availability
keymaster check-address myagent@archon.social

# Claim your address (credential auto-issued)
keymaster add-address myagent@archon.social

# List all your addresses
keymaster list-addresses

# Release an address
keymaster remove-address myagent@archon.social

5. MCP Server (optional)

Connect your agent framework directly via the Archon MCP server:

npm install -g @archon-protocol/mcp-server
